title: "Configuration"

<Steps>
<Step title="Enter your API Token">
  After [installing the plugin](/vatsys/installation), the first step is to add your API token from the [account page](https://preview.stopbars.com/account). This authenticates your requests and is required before you can proceed to any other configuration.
  
  ![BARS Config](/images/v2/vatSys_BARS-Config.png)
  
  <Danger>
  Keep this token secret, never share it with anyone. It is used to authenticate API requests.
  </Danger>
</Step>

<Step title="Add Airport ICAO">
  Once your API token is saved, you can add airports to manage. Enter the airport's ICAO code in the input field and click **Add**.
  
  <Tip>
  You can add multiple airports to manage them all from the Config window, particularly useful when providing topdown coverage to multiple airports.
  </Tip>
</Step>

<Step title="Configure Based on Profile Type">
  When you click **Add**, the appropriate window will automatically open based on your airport's profile type:
</Step>

</Steps>

  ### INTAS Airports

  After clicking **Add**, the airport lighting window opens immediately, ready for you to manage lighting configurations. No additional setup is required.

  ### Legacy Airports

  Legacy airports require a runway selection process before you can begin managing lighting. Here's how it works:

  <Steps>
  <Step title="Add Your Airport">
    Enter the airport ICAO code and click **Add**:

    ![Adding Legacy Airport](/images/v2/vatSys_Legacy-AirportAdd.gif)

    The airport will then auto populate within the Config window.
  </Step>

  <Step title="Runway Profiles Window Opens">
    The Runway Profiles window will automatically open for your added airport:

    ![Runway Profiles Window](/images/v2/vatSys_Legacy-RunwayProfiles.png)

    If it doesn't open, manually access it by clicking the **"PROFILES"** button.
  </Step>

  <Step title="Select Your Runways">
    Choose which runway profiles you want to manage by selecting them in the window:

    ![Selecting Runway Profiles](/images/v2/vatSys_Legacy-RunwayProfilesSelection.gif)
  </Step>

  <Step title="Dedicated Runway Windows">
    Once you've selected your desired runways, each runway will get its own dedicated lighting management window. You can now control lighting for each runway independently.
  </Step>
  </Steps>

You've now completed the installation and configuration of the plugin. Before you start controlling with BARS, we recommend learning about the different profile types, and how they work, as this will help you understand how to effectively use the plugin.
