## About

The vatSys Plugin contains two different profile types for managing airport lighting in vatSys, <Tooltip tip="Integrated Tower Automation Suite">INTAS</Tooltip> and Legacy. Both are designed and based on real-world Australian Air Traffic Control software.

## Types

In the real world, <Tooltip tip="Integrated Tower Automation Suite">INTAS</Tooltip> is far more advanced and part of a complex comprehensive automation suite. In the BARS vatSys Plugin, INTAS has been replicated to include only the essential airport lighting management features. 

INTAS is a more advanced suite with additional features including windsock indications, lead-on light animations, and reactivation timers.
Legacy is a far more simple system, which represents the design found in some older towers and contains the core airport lighting management features but excludes these additional elements.
