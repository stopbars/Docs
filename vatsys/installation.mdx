This quickstart guide shows you how to install and setup the BARS vatSys Plugin in minutes.
After completing this guide, you will have successfully installed the plugin, ready for network use.



<Info>

**Prerequisites**: 

Before you begin, you must have the following:
1. BARS Account & API Token
2. BARS Installer
3. vatSys Installed

</Info>

<Steps>
<Step title="Install the plugin with BARS Installer">
  Launch the **BARS Installer**, choose and install the **vatSys Plugin**. 
</Step>

<Step title="Open vatSys and confirm the plugin installed">
  After install, open vatSys and select **Windows** from the menu bar. If the plugin installed successfully, **BARS** should appear as an option in the dropdown. If not, verify the installation by:

    1. Selecting **Info → About** and confirming "**BARS for vatSys**" appears in the "**Plugins**" list.
    2. Checking your vatSys Plugins folder: `C:\Program Files (x86)\vatSys\bin\Plugins`.

  <Info>
    If BARS doesn't appear in vatSys or the Plugins folder, check the installer for any warnings or errors during download, then reattempt the installation.
  </Info>
</Step>

<Step title="Connect to an ATC position">
  Select a position, and connect to the network through vatSys as you normally would. Once connected, the **BARS** window becomes selectable from the **Windows** menu.
</Step>
</Steps>
