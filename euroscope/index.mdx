# EuroScope Plugin Overview

The BARS plugin for EuroScope provides controllers with a comprehensive stopbar management system that integrates seamlessly with your existing radar displays. Control runway stopbars with precision and ensure pilots receive real-time updates through the BARS network.

## What is BARS for EuroScope?

The BARS EuroScope plugin simulates a basic lighting control panel that supports control of stopbars defined in the BARS database. When you change stopbar states, the information is immediately sent to the BARS server and forwarded to connected pilots using the BARS pilot client.

<Info>
  The plugin integrates directly with EuroScope's display system, placing
  stopbar controls exactly where they appear in real-world pilot scenery for
  accurate positioning.
</Info>

## System Requirements

- **EuroScope**: Version 3.2 or higher
- **Operating System**: Windows 7 or later
- **Network**: Stable internet connection for BARS server communication
- **BARS Account**: Valid API key from [stopbars.com](https://stopbars.com/)
